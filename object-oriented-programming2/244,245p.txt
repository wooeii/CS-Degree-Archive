1) 기본 생성자가 호출되면 잔고는 0으로 이자율은 5%로 설정하는 코드를 추가하라.
2) 만약 매개 변수가 있는 생성자가 호출되면 매개 변수로 전달된 값을 멤버 변수에 할당하라.
3) 예금 계좌에 입금과 출력을 수행하는 deposit()와 withdraw() 멤버 함수를 구현하여 보라.

class BankAccount {
private:
	int balance;
	double rate;
public:
	BankAccount()
	{
		balance = 0;
		rate = 5;
	}
	BankAccount(int a, double b)
	{
		balance = a;
		rate = b;
	}
	void deposit(int money);
	void withdraw(int money);
};

void BankAccount::deposit(int money)
{
	balance = balance + money;
	withdraw(money);
}
void BankAccount::withdraw(int money)
{
	cout << money << "원 입금하였습니다." << endl;
	cout <<"잔고에 " << balance << "원이 있습니다." << endl;
}
